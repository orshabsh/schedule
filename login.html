Система «Школьное расписание»
Разработчик: Костянко Д.А. · Витебск, 2026

Вход

Расписание доступно публично. Вход нужен только для редактирования.

Как это работает

  1. Сайт размещается на GitHub Pages (статический хостинг).
  2. Данные хранятся в Supabase (Postgres + Auth).
  3. Публичный доступ — только чтение; редактирование — после входа (RLS в БД).

Если Supabase ещё не настроен, можно временно открыть dashboard и работать с демо-данными локально.
