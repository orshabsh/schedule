Как это работает

  1. Откройте login.html.
  2. На dashboard нажмите Открыть базу и выберите файл db.json.
  3. Сохранение идёт обратно в выбранный JSON (через File System Access API в Edge/Chrome).

Если браузер не поддерживает запись в файл (или вы открыли страницу не через HTTPS/localhost), можно скачать JSON вручную кнопкой «Экспорт JSON».
